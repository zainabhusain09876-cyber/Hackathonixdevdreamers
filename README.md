# Hackathonix Team - Devdreamers

## Problem
Build a Financial Goal Tracker website

## Solution
Building a modern AI-powered financial goal tracking web application.

## Features
- Set financial goals
- Track savings progress
- Monitor expenses
- View analytics
- Receive AI-powered financial recommendations

## Tech Stack
Frontend: 
- React
- TypeScript
- Tailwind CSS
- Lucide Icons
- Framer Motion for animations
- Recharts for data visualization

Backend:  
- Supabase (Auth & Database)
- Edge Functions for AI Analysis

## Future Improvements
- UPI transaction imports
- CSV bank statement upload
- Stocks, Mutual funds, Crypto, SIP investments
- Create a financial score for users
- App Generation
