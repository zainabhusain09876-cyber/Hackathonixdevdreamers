# Hackathonix Team - Devdreamers

## Problem
Build a Financial Goal Tracker website

## Solution
Building a modern AI-powered financial goal tracking web application.

## Features
- Set financial goals
- Track savings progress
- Monitor expenses
- View analytics
- Receive AI-powered financial recommendations

## Tech Stack
Frontend: 
- HTML
- CSS
- Vanilla JavaScript
- Tailwind CSS
- Chart.js for charts
- GSAP / AOS for animations

Backend:  
- Node.js
- Django
- Supabase
- PostgreSQL database

Other Tools: 
- OpenAI
- dyad
- GitHub

## Future Improvements
- UPI transaction imports
- CSV bank statement upload
- Stocks, Mutual funds, Crypto, SIP investments
- Create a financial score for users (Example score calculation, Savings rate, Debt ratio, Emergency fund level)
- App Generation
